Melhor repositório de links para devs

Este site é dedicado a facilitar a vida dos programadores que criam seus próprios projetos, com uma variedade de links para serem utilizados durante esse processo de criação.

Freepik

Imagens gratuitas com limite diário de 10 dowloads por conta e 3 sem conta, uma melhor busca com o uso do inglês e precisa dar crédito aos autores.

Shopity

Um banco de imagens com uma pequena variedade de acervo, que não possui limites de download, e para ter acesso não é necessário criar uma conta. É possível escolher a resolução da imagem que você quer baixar.

Pixabey

Grande variedade de imagens e vídeos 100% gratuitos com a possibilidade de escolha para qualidade.

Picwish

Funções de desblur e retirar fundo gratuitas com conta no site. Não recomendado baixar o aplicativo.

Erase.bg

Não precisa de conta para utilizar e é rápido para remover os fundos. Perfeito para fazer tudo em 10 segundos.

Adobe Express

Com várias funcionalidades além de remover fundo, precisa ter uma conta no Behance para utilizar.

Anime.js

Tem disponibilidade de frames que facilitam na hora de criar sua própria animação em JavaScript. Além disso é possível ver algumas animações já prontas e seus códigos.

Magic Animations CSS

Disponibiliza uma variedade de animações, e ao escolher e clica-la é possível ver como funciona e se desejado copiar o comando.

Zdog

Possibilita a criação de modelos de animação 3D, ele possui disponibilidade de alguns modelos, onde é possível visualizar o código.

Hover.css

Tem uma grande variedade de hovers para que possa escolher o que atende a sua necessidade. É possível acessar o código baixando o hover.css ou acessando o repositório no GitHub.

Font Library

Mostra como cada caractere fica na fonte e como aplicar a fonte em HTML, além de ter como baixar as fontes gratuitamente.

DaFont

Grande variedades de fontes disponíveis de forma totalmente gratuita, e logo abaixo do nome tem esclarecido sobre o direito de uso.

Netfontes

Fontes grátis com variedade e sem limites pra baixar , também permite escolher o tamanho.

Befonts

Fontes diversificadas, há filtro para fontes gratuitas e pagas, e especificação de licença para uso.

Free Best Fonts

Grande variedade de fontes, totalmente gratuito, contém especificação de licença para uso.

Edx

Oferecem cursos online gratuitos de alta qualidade com tutores de grandes universidades para estudantes em todo o mundo.

Coursera

Em inglês, com artigos sobre vários assuntos de programação, mas com pesquisa confusa e com outros artigos misturados.

BitDegree

Site todo em inglês, conceitos básicos sobre várias linguagens de programação e HTML, CSS, possibilidade de testar o código e observar na prática como funciona, interface para teste parecida com CodePen

Beecrowd

Site para treinar programação e banco de dados com diversos exercicios em portugues e ingles.

Coolors

São oferecidas paletas prontas ou a possibilidade de gerar novas. Pra isso, existe a ferramenta seletor de imagens, lá você seleciona uma imagem sua ou do próprio site e a partir dela será gerada uma paleta.

Adobe ID

O site gera paletas de forma gratuita pelo método que você escolher (monocromático, análogo, etc.). Neles, você pode extrair o tema e o gradiente de uma imagem, fazer uma verificação de contraste e até pesquisar uma palavra para conseguir paletas relacionadas a ela - como "céu", por exemplo.

Color Hunt

O site oferece paletas prontas, você pode baixa-las, copiar a numeração de uma cor especifica e salvar a paleta dentro do próprio site curtindo, também é possível criar sua própria paleta de cores.

Color Lisa

Paleta de cores dos maiores artistas do mundo para você se inspirar e criar a sua.
